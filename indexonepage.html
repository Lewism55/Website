React App: Food Now

X

Food Now



Food Now is a search app I have created that uses the Yelp API to allow users to search for restaurants and food providers in a number of locations including the UK and USA. The aim was to make an app with a simple search bar and some sorting filters. For each restaurant there is a brief info section and a link to the Yelp page for directions etc.

The app contains 5 components with various event handlers and listeners. The final build was deployed using Node.js.

Languages/tech used:
-HTML
-CSS
-Javascript
-React
-Node.js

App Functions:

-Clickable links to yelp on each image.

-3x sort by filters which allow reordering.

-Search returns using the “enter” keypress or clicks.

-A dual search bar using type and location criteria.

-Optimised styling for all screen sizes (incl. mobile).

React App: Quick A Track

X

Quick A Track


(please note: to access the app you will need to use a spotify login. Alternatively, there is a video below showing the app functionality)

Quick A Track is an app designed to allow users to create a playlist and upload it to their Spotify account. Currently playlist creation on Spotify is a laborious process which this app steamlines.

The app contains 7 components. Firstly you can search for songs via artist, album or song name which returns a list of results. These songs can individually be added or removed from a new playlist The user can then customise the new playlist name and save it to Spotify. The app will then create this new playlist in the user's Spotify account. I deployed this app using Surge to trial a different build and deployment method.

Languages/tech used:
-HTML
-CSS
-Javascript
-React

App Functions:

-A search bar allowing album, artist or song name requests

-Search returns using the “enter” keypress or mouse clicks.

-An independent scrollable results list

-A playlist list with a customisable title

-A simple button press to add or remove songs from a playlist

-A save to Spotify ability which live updates in the user’s chosen account.

-Optimised styling for all screen sizes (incl. mobile)

Wireframing

Whilst learning the basics of HTML and CSS I thought it would be useful to look at the design process. As part of this, I tested out a few different wireframing tools such as inVision, Axure and Balsamiq. When my website was still in it's infancy I drew up some basic sketches on paper, then tried to design these in order to have a design to work from when I started styling my web pages.
